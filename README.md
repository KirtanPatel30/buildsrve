# BuildServe — Building Maintenance Portal

A web application that helps apartment residents report and track building maintenance issues, and helps building managers organize and respond to them.

## How to Run Locally

1. Make sure you have Node.js installed (version 14 or newer)
2. Clone this repository
3. Open a terminal in the project folder and run:

```
npm install
npm start
```

4. Open your browser and go to: http://localhost:3000
